## 一、技术栈

```
浏览器 (PWA)
    │  http://localhost:3009
    ▼
Next.js 14 (TypeScript)    FastAPI + RapidOCR    PostgreSQL 16    DeepSeek API
├─ 6前端页面              ├─ DOCX/PDF/图片解析    ├─ 10张表        ├─ v4-pro (问答)
├─ 24个API路由            ├─ 扫描版PDF自动OCR    ├─ script_versions ├─ v4-flash (分类)
├─ JWT中间件 (jose)       └─ ONNX Runtime驱动     └─ qa_cache      └─ 压缩/摘要
└─ Service Worker (PWA)
```

### 部署
- Docker Compose 3容器: nextjs(:3009) + postgres(:5432) + fastapi-ai(:8000)
- Git 版本控制 (25 commits)
- 自动化测试: `bash test_features.sh`

## 四、认证架构

```
请求 → middleware.ts (jose JWT验证)
     → 白名单 /api/auth/login: 跳过
     → 其他所有 /api/*: 验证 session_token cookie
     → 注入 x-user-id / x-user-role header
     → 路由处理器通过 getUser() 读取
```
