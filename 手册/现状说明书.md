## 六、FastAPI文档解析服务

### 技术栈
- Python 3.12 + FastAPI + RapidOCR (ONNX Runtime)
- 替代了原有的 PaddleOCR（PIR模型格式与OneDNN不兼容）

### 文档解析能力

| 格式 | 方法 | 说明 |
|------|------|------|
| .docx | 直接提取 | python-docx解析 |
| .pdf (文字版) | PyMuPDF提取 | 嵌入文字≥50字符 |
| .pdf (扫描版) | **自动OCR** | 文字<50字符→逐页渲染→RapidOCR |
| .png/.jpg | RapidOCR | 直接识别中英文 |

### OCR引擎：RapidOCR
- ONNX Runtime驱动，无C++框架兼容性问题
- 模型：PP-OCRv5 (det+rec) + Chinese recognition
- 扫描PDF：DPI 150，逐页渲染+识别，控制台输出进度
- 超时：PDF 10分钟，图片 2分钟
- Docker镜像预下载模型，首次调用无需等待
