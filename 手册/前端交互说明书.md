## 整体架构

```
middleware.ts          — JWT 全局认证 (Cookie验证, 注入header)
layout.tsx            — 全局布局 + PwaProvider (SW注册 + 离线检测 + 30min登出)
├─ page.tsx           — 主页·对话
├─ scripts/page.tsx   — 剧本管理
├─ adapt/page.tsx     — 改编工坊
├─ employees/page.tsx — 员工管理
├─ schedules/page.tsx — 排班管理
├─ me/page.tsx        — 个人信息
├─ dashboard/page.tsx — 数据仪表盘
└─ kiosk/page.tsx     — Kiosk大屏
```

**认证**: 所有 API 请求经 JWT 中间件, Cookie 自动携带, 前端不传 `x-user-data`
**刷新**: 全页面 `visibilitychange` 自动刷新
**PWA**: SW 缓存静态资源, 可安装到手机, 离线回退页

---

## 跨页面通用规则

- **按钮禁用**: 操作中 disabled + 视觉变灰 + 文案切换（"保存中..."/"AI分析中..."）
- **错误处理**: 红色内联横幅, 不用 alert
- **删除确认**: `confirm()` 原生对话框
- **401处理**: 清除状态显示登录框

## 页面一: 主页 `/`

### 剧本概览卡片
- 标题 + 敏感本标签
- 属性标签: 类型(紫) / 人数(蓝) / 幕数(绿) / 难度(橙) / 时长(灰)
- 角色按钮: 点选变紫切换视角, 再次点击取消
- 快捷提问按钮(6个): 凶手/关键线索/DM开场白/时间线/角色关系/动机分析

### 对话搜索
- ≥2字符触发, 防抖300ms, ILIKE中文全文
- 结果高亮 `<mark>` 标签, 跨剧本自动跳转

### 引用溯源
- AI回答中的 `【第X幕·角色】` 转为黄色可点击链接
- 点击弹出原文上下文弹窗

### 断点续传
- SSE中断后显示黄色按钮 "⚡ 回答中断，点击继续"
- 点继续后AI从断点续写

### 登录页
- 工号/手机号 + 密码, bcrypt验证
- 未登录时不请求API, 控制台无401报错

## 页面二: 剧本管理 `/scripts`

### 异步上传
- 文件秒收, 弹窗1秒自动关闭, 后台5路并发解析
- 解析进度徽标: 黄色闪烁 "15/40 已解析" / 绿色 "✓ 全部就绪"
- 每5秒自动轮询进度

### 拖拽上传
- 文件夹递归遍历, "选择文件"/"选择文件夹"按钮
- >500MB自动拒绝, 格式不支持自动标记
- MD5三级去重 (批次内→数据库→跨批次)

### AI自动分类
- 一键分析文件名 → 自动填入类型+角色名 → 用户可修正

## 页面三: 改编工坊 `/adapt`

### 两步改编
- Step1: 填写指令 → 预览改编方案 (黄色卡片)
- Step2: 确认 → 生成全本 (DB事务保存)

### 差异高亮
- 点"对比原版" → 统一视图, 行号标注
- 绿色=新增行, 红色=删除行, 顶部显示统计

### 版本历史
- 每次改编自动保存原版+改编版快照
- 版本列表: 标签(原版/改编/恢复) + 预览 + 恢复按钮
- 恢复操作: confirm → 覆盖缓存 → 新快照

### 导出
- DOCX格式, 自动添加"改编版本 · 仅供内部使用"水印
