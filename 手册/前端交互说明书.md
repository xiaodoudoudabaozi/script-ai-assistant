# 剧本杀AI店员助手 — 前端交互说明书 v2.3

> 2026-05-01

---

## 整体架构

```
middleware.ts          — JWT 全局认证 (请求拦截层)
layout.tsx            — 全局布局 + PwaProvider
├─ page.tsx           — 主页·对话
├─ scripts/page.tsx   — 剧本管理
├─ adapt/page.tsx     — 改编工坊
├─ employees/page.tsx — 员工管理
├─ schedules/page.tsx — 排班管理
└─ me/page.tsx        — 个人信息
```

**认证**: 所有 `/api/*` 经 JWT 中间件, 未登录返回 401
**刷新**: 全页面 `visibilitychange` 自动刷新
**PWA**: SW 缓存静态资源, 离线回退页

## 页面一: 主页 `/`

### 状态变量 (22个)
user, scripts, scriptsLoading, selectedScript, messages, input, isLoading, error, search, conversationId, showScriptPicker, scriptSearch, historyLoading, characterName, characterList, searchResults, searching, showSearchResults

### 选剧本流程
```
"+ 选择剧本" → 弹窗(搜索/类型标签) → 选剧本
→ 有已有对话? → 加载历史 + 角色列表
→ 无对话? → 创建新对话 + 加载角色列表
```

### 剧本概览卡片 (v2.3)
- 标题 + 敏感本标签
- 属性: 类型(紫) / 人数(蓝) / 幕数(绿) / 难度(橙) / 时长(灰)
- 角色按钮: 点选变紫, 再点取消, 显示"仅看XX视角"
- 快捷提问 (6个): 凶手是谁 / 关键线索 / DM开场白 / 时间线 / 角色关系 / 动机分析
- 点按钮自动发送, 回答中全部禁用

### 搜索对话历史
- 关键词 ≥ 2字符, 防抖 300ms
- ILIKE 中文全文, 匹配内容高亮 `<mark>`
- 结果: 剧本名标签 + 对话标题 + 高亮片段
- 点击跳转到对应对话 (跨剧本自动切换)

### 各状态
| 状态 | 表现 |
|------|------|
| 未登录 | 居中登录卡片 |
| 剧本加载中 | "加载中..." |
| 空剧本 | "暂无可选剧本" |
| 对话加载中 | "加载对话历史..." |
| AI回复中 | "回答中..." + ●●● + textarea禁用 |
| 搜索中 | 搜索框右侧"搜索中..." |
| 缓存命中 | 瞬间返回 (X-QA-Cache: hit) |

## 页面二: 剧本管理 `/scripts`

### 异步上传 (v2.3)
- 文件秒收 → 弹窗 1s 自动关闭
- 后台并发解析 (5路)
- 解析进度徽标: 黄色闪烁 "15/40 已解析" / 绿色 "✓ 全部就绪"
- 自动轮询 5s

### 拖拽上传
- 文件夹拖拽递归遍历
- "选择文件" / "选择文件夹" 按钮
- >500MB / 格式不支持 → error 标记
- MD5 三级去重

### 各状态
| 状态 | 表现 |
|------|------|
| 加载中 | 居中文字 |
| 空列表 | "暂无剧本" |
| 上传中 | 进度条 + 按钮文案 + 全表单禁用 |
| OCR解析中 | 后台静默, 徽标显示进度 |
| 全部解析完成 | "✓ 全部就绪" |

## 页面三: 改编工坊 `/adapt`

### 改编流程
- Step1: 填表单 → "生成改编方案" → 黄色预览
- Step2: "确认并生成全本" → 结果展示
- 导出: DOCX (自动水印)
- 对比: 左右双栏 (原版 vs 改编版)

### 版本历史 (v2.3)
- 选剧本自动加载版本列表
- 每版本: 标签(原版/改编/恢复) + 预览按钮 + 恢复按钮
- 预览: 黄色区域显示全文
- 恢复: confirm → 覆盖当前缓存 → 新版本快照
- 改编生成时自动保存原版+新版本

### 各状态
| 状态 | 表现 |
|------|------|
| 生成中 | "AI分析中..." / "AI改编中..." + 全表单禁用 |
| 预览就绪 | 黄色背景块 |
| 截断 | 红色警告 |
| 对比模式 | 左灰原版 右紫改编版 |
| 版本预览 | 黄色区域 (可滚动) |
